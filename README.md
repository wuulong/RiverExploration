## 📖 專案簡介

《流域導航》是一部結合 **地貌學、地名政治、技術政治學** 與 **生成式 AI (GenAI)** 技術的實驗性地誌作品。我們試圖撥開水泥堤防與官方敘事的迷霧，帶領讀者看見台灣流域背後的靈魂座標。

本專案的核心在於提出一套「人、河、AI 三位一體」的探索範式。除了精闢的流域解析，更提供了一套可實踐的工作流，賦予現代探索者「數位導航」與「地景解碼」的能力。

## 🌟 核心特色

- **深度解讀模型**：不談網美景點，專注於「地學宿命」、「權力架構」與「文化復振」的深度批判。
- **二階段研究注入**：內容結合了橫向跨域資料與針對特定流域的深度採掘（Deep Research），具備極高的資訊厚度。
- **WalkGIS 數位神經系統**：整合 [WalkGIS](https://github.com/wuulong/walkgis) 工具，推動「公民參與、集體共創、迭代豐富」的自生長圖資。
- **在地化靈魂**：全書採用台灣在地語彙修正，注入「咱的土地、咱的河流」的主體意識。

## 🗺️ 內容目錄 (TOC)

1.  **第一部分：探索基本功** —— 閱讀河流的方法論。
2.  **第二部分：流域導航索引** —— 台灣 11 門靈魂流域的深度解碼。
3.  **第三部分：動態實境錄檔案櫃** —— 導航者的實地探訪與批判思考紀錄。
4.  **第四部分：未來導航** —— 當 Rivers 遇見 AI（AI 擴增探索工作流）。

## 🛠️ 數位工具箱 (Appendix)

本專案於 `Appendix_A.md` 提供了豐富的 **Prompt 集錦**，包含：
- 地景解構指令
- 地名政治挖掘指令
- 語音筆記自動化處理 (RiverLog Processing)

## 📁 目錄結構說明

- **`Research/`**：本專案的核心知識庫。包含：
    - **原始研究素材**（如：台灣十一大流域跨域研究素材）：寫作初期的橫向掃描資料。
    - **流域深度報告**（如：[N].某某溪流域研究重點分析）：針對特定流域進行深度採掘 (Deep Research) 後產出的高品質研究全文。
    - **AI 寫作對話紀錄** (`AIQA-《流域導航...》.md`)：詳實記錄了本書產製過程中，與 AI 進行深度探索 (Deep Research) 的原始 Prompt 與問答歷程，是「二階段寫作法」的實戰紀錄。
- **`FieldLogs/`**：導航者的實地探勘紀錄，是書本內容的感性實踐與動態擴充。
- **`Appendix_A.md`**：搜集並整理了全書使用到的高效 AI 指令集。
- **`RiverExploration_FullBook.md`**：為方便 AI 讀取或電子書製作，將全書按順序合併而成的單一合訂本檔案。

## 🤝 如何貢獻

本專案採用 **「動態檔案櫃」** 架構。我們邀請您：
1.  **提交實境錄 (Field Logs)**：將您的實地探索經驗，按照專案規範寫成 Log 並提交至 `FieldLogs/` 目錄。
2.  **豐富 GIS 圖資**：透過 WalkGIS 標記隱藏溪流或消失遺跡，並回饋至數位地誌庫。
